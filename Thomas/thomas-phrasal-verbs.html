Separable vs Inseparable
Where the object can go
›
Transitive phrasal verbs (they take an object) are either separable or inseparable. Intransitive ones take no object at all.
🇫🇷 Les verbes à particule transitifs sont séparables ou inséparables ; les intransitifs n'ont pas d'objet.
TypeRuleExample
Separableobject between verb + particle; must go there if it's a pronounSet up the stage. → Set it up.
Inseparableobject always after the particleRun into a problem. → Run into it.
Intransitiveno object at allThe festival kicked off.
⚠️ Classic mistakes
"Set up it" → "Set it up"Pronoun object must split a separable verb
"Ran it into" → "Ran into it"Inseparable — object never splits the verb

What the Particle Adds
up, out, down, off — common meaning patterns
›
Particles aren't random — the same one often adds a similar shade of meaning across many verbs. Spotting the pattern helps you guess new phrasal verbs.
🇫🇷 La particule ajoute souvent un sens récurrent, ce qui aide à deviner de nouveaux verbes.
ParticleCommon patternExamples
upcompletion / increasewrap up, fill up, wind up
outdiscovery / disappearancefigure out, sort out, cut out
downreduction / stoppingwind down, break down, calm down
offseparation / departurehead off, set off, cut off
oncontinuationcarry on, keep on, hold on
backreturn / reversalturn back, get back, pay back

Register
Phrasal verbs are informal — know the formal swap
›
Phrasal verbs are the natural, spoken choice. In writing or formal speech, a single-word verb often replaces them.
🇫🇷 À l'oral, on utilise le verbe à particule ; à l'écrit formel, on préfère souvent un seul mot.
EverydayFormal equivalent
sort outresolve
figure outdetermine
come up withdevise
put offpostpone
look intoinvestigate
