Polite Requests

I'd Like…, Could I Have…?

📘 1. Understanding the Rule

Saying what you want directly (I want…) can sound rude in English. These softer patterns get you the same thing — politely.

wanting something
I'd like a coffee, please.
asking for an object
Could I have the menu, please?
asking someone to act
Could you open the window, please?
offering something
Would you like some water?
Careful: "I want" sounds blunt. I want a coffee is grammatically correct but sounds demanding. I'd like a coffee is the everyday polite version — always add "please".

📊 2. Reference Tables

Core patterns

PatternUse
I'd like… (= I would like)saying what you want, politely
Can/Could I have…?asking for an object
Can/Could you…?asking someone to do something
Would you like…?offering something to someone

Politeness scale

LevelExample
directGive me the salt.
everyday politeCan I have the salt, please?
more politeCould I have the salt, please?
very polite / formalWould it be possible to have the salt?

18 exercises across all six CEFR levels — three per level, increasing in difficulty. Answers on the last page.

LEVEL A1 · Beginner
Exercise 1 — Multiple choice: circle the more polite option.
  1. (Give me water. / I'd like some water, please.)
  2. (Can I have the menu? / Menu now.)
  3. (I want a tea. / I'd like a tea, please.)
  4. (Could I have the bill, please? / Bill!)
  5. (Open the door. / Could you open the door, please?)
  6. (Would you like some cake? / You eat cake.)
Exercise 2 — Matching: match each situation (1–4) to the polite request (a–d).
1. In a café
a. Could you pass the salt, please?
2. At a friend's dinner table
b. I'd like a coffee, please.
3. In a shop, trying on clothes
c. Could I try this on, please?
4. On the phone
d. Could I speak to Anna, please?

  3. ______ the bill, please?
  4. ______ a glass of water, please.
  5. ______ some more time to decide?

LEVEL B2 · Upper-Intermediate
Exercise 10 — Read, then mark each statement T or F.

"I was wondering if it would be possible to move our meeting to Thursday — would that work for you? I know it's short notice, so I'd completely understand if you couldn't. If Thursday's difficult, would you mind letting me know a time that suits you better? I'd really appreciate it."

  1. The speaker is asking to reschedule a meeting. ___
  2. The speaker insists on Thursday no matter what. ___
  3. The speaker offers an alternative if Thursday doesn't work. ___

Exercise 12 — Guided writing.

Write a short polite email (6–8 sentences) asking a colleague for a favour, using at least three different politeness patterns from this page (I'd like, Would you mind, I was wondering if…).

LEVEL C1 · Advanced
Exercise 13 — Rewrite each request in the most formal, hedged register possible.
  1. Can I have an extension?
  2. Could you send me the file?
  3. I want a refund.
  4. Can you call me back?
  5. Give me some feedback on this.
Exercise 14 — Multiple choice: what does the extra hedging achieve here?
  1. "I don't suppose you could help me, could you?" a) very indirect, minimising imposition  b) rude
  2. "If it's not too much trouble, could you…?" a) softening the request  b) making it more urgent
  3. "I hate to ask, but…" a) acknowledging the imposition  b) demanding
  4. "Might I trouble you for…?" a) very formal, old-fashioned politeness  b) casual
Exercise 15 — Correct the mistakes in these complex, hedged requests.
  1. I don't suppose you could to help me?
  2. Would it be possible for you sending the file?
  3. I was hoping if you could join us.
  4. Might I to trouble you for a moment?
  5. If it wouldn't be too much trouble, could you to call me?
LEVEL C2 · Proficiency
Exercise 16 — Open writing.

In 80–120 words, write a formal email to a senior client asking for a delayed payment to be resolved, using layered hedging (I was wondering whether, might it be possible, I'd be most grateful if) to stay polite while being clear.

Exercise 17 — Reorder into formal, complex requests.
  1. grateful / would / I / be / most / if / you / could / respond
  2. wondering / I / possible / it / whether / would / was / be / to / reschedule
  3. trouble / might / for / I / a / moment / you
  4. too / not / trouble / it / much / were / if
  5. appreciate / I / would / greatly / assistance / your
Exercise 18 — Correct the nuanced errors.
  1. I would be most grateful if you would to respond.
  2. I was wondering would it be possible to reschedule.
  3. Might I to trouble you for your assistance?
  4. I would greatly appreciate for your help.
  5. If it were not too much trouble, would you to call me?
