There Is / There Are

Talking About What Exists

📘 1. Understanding the Rule

There is / there are introduces something into a conversation for the first time — it says "this exists". Once the listener knows about it, we switch to it or they to refer back.

there is
singular / uncountable: There is a cat.
there are
plural: There are three cats.
question
Is there…? / Are there…?
negative
There isn't… / There aren't…
Careful: "there is" ≠ "it is". There is a book on the table. (introducing it) It is red. (referring back to it). Don't say It is a book on the table for new information.
